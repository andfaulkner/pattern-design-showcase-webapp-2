Display types
=============

block
-----

*   always starts on a new line 
*   takes up the full width available 

*   naturally block elements:

        <div>
        <h1> - <h6>
        <p>
        <form>

inline
------

*   does not start on a new line 
*   only takes up as much width as necessary

*   naturally inline elements:

        <span>
        <a>
        <img>

----
